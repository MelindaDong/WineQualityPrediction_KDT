# kd-tree-and-forest

Wine experts evaluate the quality of wine based on sensory data. We could also collect the features of wine from objective tests, thus the objective features could be used to predict the expertâ€™s judgement, which is the quality rating of the wine. This could be formed as a supervised learning problem with the objective features as the data features and wine quality rating as the data labels.

In this project, the data provides objective features obtained from physicochemical statistics for each white wine sample and its corresponding rating provided by wine experts. 

k-d tree (KDT), and random forest were inplemented to provide wine quality prediction on the test set.
